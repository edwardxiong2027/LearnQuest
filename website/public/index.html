Built by Students, for Students

LearnQuest was created by the Diamond Bar High School Vibe Coding Club with a simple mission: make quality education free, accessible, and fun for every student, regardless of their internet access or financial situation.

We believe that every child deserves access to a patient, knowledgeable tutor that is available anytime, anywhere. By combining a lightweight AI model with a carefully crafted curriculum, LearnQuest delivers a personalized learning experience that works completely offline.

Our platform is open source because we believe in the power of community. Teachers, developers, and students around the world can contribute new content, fix bugs, and help us reach more learners.

Get Started

Everything you need to set up LearnQuest on your machine or USB drive.

System Requirements

  • ✓ Mac, Windows, or Linux
  • ✓ Python 3.8 or higher
  • ✓ 4 GB RAM minimum
  • ✓ 8 GB free disk space
  • ✓ A modern web browser
  • ✓ Internet for initial setup only

Quick Start

Step 1 — Clone the repository

git clone https://github.com/AaronFR/LearnQuest.git

Step 2 — Run setup

python launch.py setup

Step 3 — Start learning

python launch.py start
